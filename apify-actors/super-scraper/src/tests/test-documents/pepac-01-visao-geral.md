# PEPAC - Plano Estratégico da PAC
## Visão Geral do Plano

### O que é o PEPAC

O **Plano Estratégico da PAC (PEPAC)** é o instrumento de programação da Política Agrícola Comum para Portugal no período 2023-2027.

### Dotação Financeira

| Pilar | Fundo | Montante (M€) |
|-------|-------|---------------|
| Pilar I | FEAGA - Pagamentos Directos | 4.400 |
| Pilar II | FEADER - Desenvolvimento Rural | 2.200 |
| **Total** | | **6.600** |

### Objectivos Específicos

1. **Rendimento agrícola viável** - Apoiar rendimentos sustentáveis
2. **Competitividade** - Aumentar orientação para o mercado
3. **Cadeia de valor** - Melhorar posição dos agricultores
4. **Clima** - Contribuir para mitigação
5. **Ambiente** - Gestão sustentável dos recursos naturais
6. **Paisagem e biodiversidade** - Preservar habitats
7. **Renovação geracional** - Atrair jovens agricultores
8. **Emprego rural** - Dinamizar zonas rurais
9. **Alimentação** - Responder às expectativas dos consumidores

### Estrutura de Intervenções

#### Pagamentos Directos (Pilar I)
- Apoio de Base ao Rendimento
- Apoio Redistributivo
- Eco-Regimes
- Apoio Associado

#### Desenvolvimento Rural (Pilar II)
- Investimento nas explorações
- Instalação de jovens agricultores
- Medidas agroambientais
- LEADER
