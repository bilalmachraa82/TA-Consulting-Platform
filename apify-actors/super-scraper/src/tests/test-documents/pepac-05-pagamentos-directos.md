# PEPAC - Pagamentos Directos
## Apoio de Base ao Rendimento

### Estrutura dos Pagamentos Directos

O **Apoio de Base ao Rendimento para a Sustentabilidade (ABRS)** é o principal componente dos pagamentos directos do Pilar I.

### Componentes dos Pagamentos Directos

| Componente | % do Envelope | Descrição |
|------------|---------------|-----------|
| ABRS | 40% | Apoio base por hectare elegível |
| Apoio Redistributivo | 10% | Majoração para primeiros hectares |
| Eco-Regimes | 25% | Práticas ambientais voluntárias |
| Jovens Agricultores | 3% | Majoração para JA |
| Apoio Associado | 15% | Apoio à produção específica |
| Reserva | 7% | Casos específicos |

### Apoio de Base ao Rendimento (ABRS)

**Pagamento médio estimado**: 120€/ha

**Elegibilidade**:
- Superfície agrícola utilizável (SAU)
- Terras aráveis, culturas permanentes, pastagens permanentes
- Mínimo: 0,5 ha

### Apoio Redistributivo Complementar

Para apoiar explorações de menor dimensão.

**Valor adicional**: ~50€/ha  
**Aplica-se aos**: primeiros 10 ha

### Apoio a Jovens Agricultores

Majoração adicional de **25%** do ABRS para agricultores até 40 anos nos primeiros 5 anos de instalação.

### Condicionalidade Reforçada

Para receber pagamentos directos, os agricultores devem cumprir:

1. **BCAA** - Boas Condições Agrícolas e Ambientais (8 normas)
2. **RLG** - Requisitos Legais de Gestão (11 directivas/regulamentos)

#### Normas BCAA principais:
- BCAA 1: Manutenção de pastagens permanentes
- BCAA 4: Faixas de protecção ao longo de cursos de água
- BCAA 7: Rotação de culturas (terras aráveis)
